# Customer Segmentation - Data Science

This project explores customer segmentation using two approaches:

## 1. KMeans (Baseline)
Clustering based on engineered customer features.

## 2. Autoencoder + KMeans
Dimensionality reduction using a neural network before clustering.

## Key Insight
Using an autoencoder improved cluster separation and helped identify different types of customer issues:

- Delivery problems
- Payment issues
- High-value dissatisfied customers
